  - 合并多条相关日志为单条信息丰富的日志

    window[statusKey] = { progress: currentProgress, status: currentStatus };
}

// 服务状态变化检测
const hasServiceChanges = status.services_install_status.some((service, index) => {
    const lastService = lastServices[index];

- **解决方案：**

  - 只对真正新增的日志应用进入动画
