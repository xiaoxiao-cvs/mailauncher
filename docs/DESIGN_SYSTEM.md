## 使用指南

### 背景

#### 深色主题
- 主背景：`federal_blue[500]` 到 `marian_blue[500]` 渐变
- 卡片背景：白色半透明 + 背景模糊
- 装饰背景：`honolulu_blue[500]`, `blue_green[500]`, `pacific_cyan[500]`

#### 浅色主题
- 主背景：白色 (#ffffff)
- 卡片背景：白色半透明 + 背景模糊
- 动态装饰：`uranian_blue[500]`, `light_sky_blue[500]`, `thistle[500]`, `fairy_tale[500]`, `carnation_pink[500]` 渐变球

### 交互元素
- 主按钮（深色主题）：`honolulu_blue[500]` 到 `blue_green[500]` 渐变
- 主按钮（浅色主题）：`light_sky_blue[500]` 到 `carnation_pink[500]` 渐变
- 次要按钮：白色半透明边框
- 悬停状态：增加透明度或使用更浅的色阶

### 文本

#### 深色主题
- 主文本：白色 (#ffffff)
- 次要文本：白色 80% 透明度 (rgba(255, 255, 255, 0.8))
- 提示文本：白色 50-70% 透明度

#### 浅色主题
- 主文本：`federal_blue[500]` 或 `marian_blue[500]`
- 次要文本：`federal_blue[400]` 或 `marian_blue[400]`
- 提示文本：`marian_blue[300]`

### 图标
- 图标背景：使用对应的主色调渐变
- 图标颜色：白色

## 设计原则

1. 保持一致性：所有颜色必须从上述调色板中选择
2. 注重对比度：确保文本在背景上清晰可读
3. 分层使用：通过不同的蓝色色阶营造层次感
4. 渐变使用：相邻色阶的渐变效果最佳

## 禁止事项

1. 禁止在项目中使用任何 emoji 字符
2. 禁止使用调色板之外的颜色
3. 禁止使用纯黑或纯灰色（除非作为透明度使用）
