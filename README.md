# 麦麦启动器

全新重写的麦麦启动器前端项目。

## 技术栈

- **Vue 3** - 使用 Composition API
- **Vue Router** - 路由管理
- **Pinia** - 状态管理
- **Tailwind CSS 3** - 样式框架
- **DaisyUI** - UI 组件库
- **Anime.js** - 动画库
- **Vite** - 构建工具
- **pnpm** - 包管理器

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览生产构建
pnpm preview
```
